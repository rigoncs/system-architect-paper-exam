# 系统架构设计师论文备考工具

软考高级**系统架构设计师**考试论文科目备考辅助工具，包含 Claude Code 技能和考前押题范文。

## 项目结构

```
.
├── CLAUDE.md                     # Claude Code 项目指引
├── prompt.md                     # 论文写作规则（题源）
├── .claude/skills/architect-paper/  # 论文生成技能
│   ├── SKILL.md                  # 技能定义与模板
│   └── references/
│       └── cloud-native-example.md  # 云原生参考范文
└── papers/                       # 押题范文
    ├── 01-论服务网格ServiceMesh在微服务架构中的应用.md
    ├── 02-论系统可靠性设计及其应用.md
    ├── 03-论Kappa架构在大数据处理中的应用.md
    ├── 04-论AIOps在智能运维中的应用.md
    └── 05-论分布式缓存技术的应用.md
```

## 考试背景

考试第三门科目为论文，四选一，要求 2200-3000 字。采用**十段式**结构：

| 部分 | 段数 | 建议字数 |
|------|------|----------|
| 摘要 | 2段 | 300字 |
| 项目背景 | 2段 | 500字 |
| 正文 | 3段 | 1200字 |
| 结尾 | 2段 | 400字 |

论文必须以系统架构设计师的第一人称视角写作，围绕项目实践经验展开论述，要求三个分论点且需回应子题目。

## 使用方法

### 生成新论文

在 Claude Code 中直接描述你的需求，`architect-paper` 技能会自动激活。例如：

- "帮我写一篇论云原生架构设计的论文"
- "生成一篇关于系统安全性设计的押题范文"

技能会按三步流程引导你完成：确定题目 → 确定项目参数 → 生成全文。

### 背诵现有范文

`papers/` 目录下的 5 篇押题范文是综合 2024-2025 年真题趋势和多方预测生成的，可以直接背诵。优先级建议：

1. **服务网格** + **可靠性设计**（命中概率最高）
2. **Kappa架构** + **AIOps** + **分布式缓存**（补充）

## 押题依据

基于对 2024-2025 年共 4 场真题的分析，排除了最近已考的主题（事件驱动架构、Serverless、Lambda架构、负载均衡、性能测试等），结合知乎"方才coding"等多方预测选定 5 个方向。
